# 🎮 Game Backlog Tracker

[Ana Proje](`https://github.com/berkeguvenc/homework-game-backlog-tracker`)

Bu proje, React, TypeScript ve Vite kullanılarak geliştirilmiş bir Oyun Backlog Takip uygulamasıdır.

## 🚀 Projeyi Çalıştırma Adımları

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

1. **Bağımlılıkları yükleyin:**
   ```bash
   npm install
   ```

2. **Uygulamayı başlatın:**
   ```bash
   npm run dev
   ```
   *Not: Bu komut `concurrently` kullanarak hem Vite geliştirme sunucusunu hem de `json-server` backend'ini aynı anda başlatacaktır.*
